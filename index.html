PCT-100 实训平台

化工控制实训装置产品说明书

基于 ESP32 的智能化工过程控制系统，集成温度监控、光照控制、 WiFi/MQTT 远程通信，适用于高校实训教学与工业自动化培训。

产品概述

一款为化工自动化实训量身定制的综合控制平台

🌡

温度智能监控

搭载 DS18B20 数字温度传感器，精度 ±0.5°C，支持自动温控风扇启停， 阈值可通过 MQTT 远程下发。

☀

光照自动调节

光敏传感器实时采集环境光照强度（ADC），超过设定阈值自动开启 灯光继电器，模拟化工车间照明控制。

🔌

WiFi / MQTT 远程通信

ESP32 内置 WiFi 模块连接云端，通过 MQTT 协议实现远程状态上报 与指令下发，支持阈值远程配置。

💡

RGB 万色状态灯

板载 RGB LED 实时显示系统状态：WiFi 连接、MQTT 在线、 温度/光照越限报警，一目了然。

💾

双模式控制

支持"自动控制"与"手动控制"两种运行模式。自动模式下由传感器 数据驱动；手动模式下可独立控制灯与风扇。

📝

显示屏实时反馈

显示屏循环展示温度、光照 ADC 值、WiFi 状态、SSID 等关键信息， 方便实训过程中实时观察。

技术规格

PCT-100 硬件与电气参数一览

项目 参数 说明
主控芯片 ESP32 双核 240MHz，内置 WiFi & Bluetooth
温度传感器 DS18B20 数字一线总线，量程 -55°C ~ +125°C，精度 ±0.5°C
光敏传感器 光敏电阻 + ADC 模拟量采集，阈值默认 900（可通过 MQTT 配置）

风扇继电器 GPIO 7 控制散热风扇，支持自动/手动切换

操作按键 KEY1 + KEY2 自锁电源开关 + 模式切换按键（支持长按/短按）
系统架构 FreeRTOS 多线程 WiFi/MQTT 线程 + LED 线程 + 主循环线程
串口指令 115200 bps 支持 clearwifi / showwifi / reboot / mqtt / help

操作模式详解

设备支持自动与手动两种运行模式，灵活适配实训需求

自动模式 智能传感器驱动

系统根据传感器实时数据自动控制灯光和风扇，无需人工干预。

  • 光敏传感器 ADC > 阈值 → 自动开灯
  • 温度传感器 ≥ 35°C → 自动开风扇
  • MQTT 可远程下发新阈值
  • RGB LED 显示越限报警

手动模式 按键切换状态

长按 KEY2 切换自动/手动模式；手动模式下短按 KEY2 循环 4 种状态。

状态灯光风扇
00关闭关闭
01关闭开启
10开启关闭
11开启开启

串口命令参考

通过串口终端（115200 bps）可执行以下调试命令

命令功能说明

使用注意事项

⚠ 电源安全：设备使用前请确认供电电压符合规格（5V / 3.3V）， 通电状态下请勿触碰裸露的继电器触点。
⚠ 温度传感器：DS18B20 数据线需接 4.7kΩ 上拉电阻， 传感器探头避免直接接触高温液体。
⚠ WiFi 配置：首次使用需通过串口或配网模式配置 WiFi， 确保设备所在环境 2.4GHz WiFi 信号良好。
⚠ 继电器负载：继电器输出端最大负载不超过 10A / 250VAC， 连接大功率设备前请确认电气参数。
