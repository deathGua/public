项目背景

关于本项目

PCT-100 化工控制实训装置是一个面向高校化工自动化实训教学的综合平台， 旨在通过实际动手操作帮助学生理解工业过程控制的核心概念。

项目背景与意义

产教融合，培养新时代化工自动化人才

🎯 项目目标

随着工业 4.0 和智能制造的推进，化工行业对自动化控制人才的需求日益增长。 PCT-100 实训装置将工业现场常用的传感器、执行器、控制器和通信技术 整合到一个教学平台上，使学生能够在实训室环境中：

  • 理解传感器数据采集与处理的基本原理
  • 掌握继电器控制与执行器驱动的方法
  • 学习 WiFi / MQTT 物联网通信协议
  • 实践 FreeRTOS 多线程编程
  • 体验完整的工业自动化闭环控制流程

💡 技术栈

ESP32 FreeRTOS Arduino MQTT WiFi DS18B20 HTML/CSS/JS

硬件平台基于 ESP32 微控制器，采用 FreeRTOS 多线程架构。 软件方面涵盖嵌入式 C/C++ 编程（Arduino 框架）、 MQTT 物联网通信协议。网页前端使用原生 HTML + CSS + JavaScript 构建， 部署于 GitHub Pages。

学习目标与收获

通过 PCT-100 实训，预期达到的学习效果

🔧

硬件实践能力

掌握传感器接线、继电器驱动、GPIO 控制等嵌入式硬件基础技能。

💻

嵌入式编程

学习 Arduino 框架下的 C/C++ 编程，理解 FreeRTOS 多线程并发模型。

🌐

物联网通信

实践 WiFi 配网、MQTT 协议、JSON 数据格式、远程监控等 IoT 核心技术。

📊

控制理论应用

将课本中的闭环控制、阈值判断、PID 控制等理论应用到实际系统中。

项目开发历程

PCT-100 版本迭代记录

v1.9 — 万色灯优化 + 升级版 MQTT

优化 RGB LED 状态显示，升级 MQTT 通信模块，支持更丰富的状态上报。

v2.0 — 多线程模式

引入 FreeRTOS 双线程架构：WiFi/MQTT 线程 + LED 线程，实现并发处理。

v2.1 — MQTT 阈值下发

支持通过 MQTT 远程配置温度阈值和光照阈值，无需重新烧录固件。

v2.2 — 代码优化，防重启

修复系统稳定性问题，增加异常处理与看门狗机制，提升运行可靠性。

v2.3 — MQTT 阈值控制优化

优化配置管理器，实现阈值的持久化存储与动态加载，当前最新版本。
